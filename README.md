<h1 align="center"> Convolution Visualizer </h1>

## Usage ##
### Ensure all python dependencies are installed using: ### 
`pip install -r requirements.txt` 
### Run visualizer using: ###
`python start.py` 
### Follow prompts on the terminal ###
